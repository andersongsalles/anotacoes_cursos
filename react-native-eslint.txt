**** Instalacao do EsLint

1.pacotes de dependencia - npm info "eslint-config-airbnb@latest" peerDependencies
2.inserir no arquiv package.json em devDependencies:
    "eslint": "^4.9.0",
    "eslint-plugin-import": "^2.7.0",
    "eslint-plugin-jsx-a11y": "^6.0.2",
    "eslint-plugin-react": "^7.4.0"
3.Rodar o comando - npm install    
4.Instalar mais pacotes - npm i --save-dev babel-eslint eslint-config-airbnb eslint-plugin-react-native
